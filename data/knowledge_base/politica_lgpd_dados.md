# POLÍTICA CARE PLUS: LGPD E PRIVACIDADE DE DADOS
No ecossistema Blua, dados de saúde são classificados como Dados Pessoais Sensíveis.
O BluaDiagnostics (IA) atua com anonimização. Os logs de conversa e sintomas relatados pelo beneficiário não podem ser usados para treinar modelos de linguagem (Zero-Retention Policy). Os dados são estruturados apenas para compor o prontuário que será avaliado pelo médico.
